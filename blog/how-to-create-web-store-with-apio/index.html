How to Create Web Store with Chopin and Waitress

2020-07-08 • 3 min read

chopin   ecommerce   google   google-sheet   telescope   tutorial   waitress   website

Chopin by apio is a simple online store creator that allows you to configure your online store in just a few clicks. In this post, we’ll show you how you create an online store with google sheet using Waitress (apio’s google sheet API microservice). Try now and start selling your products online in just a few clicks.

Read more: How does Chopin compare to website builders and ecommerce services.

Step 1: Create Catalog Google Sheet ¶

Create a google sheet that you’ll be using as your catalog. Copy and paste the following columns to the sheet:

name nickname description product_url image_url price max_qty category

Fields are defined as the following:

  • name: unique name/id for each product
  • nickname: item names that will appear on the store page
  • description (optional): product description
  • product_url (optional): link to product page
  • image_url: url of product image
  • price: product price
  • max_qty: maximum quantity that one customer can buy. If max_qty = 0, it will show ‘Out of Stock’.
  • category (optional): to allow customer to filter products

Step 2: Authorize Catalog Google Sheet on Telescope ¶

Go to Telescope(apio microservices platform). Log in into your existing google account.

Then go to Waitress and follow the instruction in the configure tab to authorize your google sheet. After authorization, only you can access your sheet through our API.

Step 3: Create Order Google Sheet ¶

Repeat Step 1 and Step 2 to create another google sheet with different header. This sheet will be storing new order that customers submitted on your store.

Copy and paste the following columns to the sheet:

order_number user_name user_email user_phone user_address order_created order_subtotal order_shipping_option order_shipping order_tax_rate order_tax order_total product_name product_qty product_price status_open status_paid status_shipped

Step 4: Create the Store ¶

After configuring both catalog and order in google sheet, now we can head to Chopin to setup our store. Enter the store name, logo url, catalog and order APIs.

Note: In this example, we use Waitress google sheet API as our catalog and order API. Chopin will be making requests to: https://trampoline.apiobuild.com/router/waitress/gsheets/<your catalog and order google sheet id>.
Google sheet id can be extracted from google sheet url: https://docs.google.com/spreadsheets/d/<sheet-id>.

Step 5: Start Selling ¶

Voila, the store is now available at: https://tinyurl.com/yarc2454. You can use services like https://tinyurl.com/ to make the raw url (https://trampoline.apiobuild.com/router/chopin/store/page/google-oauth2%7C117090713962028193035/56422174-c3bf-4e2e-9524-2ad542d84562) easier to share on facebook, instagram, and you friends.

Any new order will be updated on the order google sheet you created in Step 3. You can also setup google sheet notification to get real-time updates on new order submission.

Happy “chopin”! 🛍️
